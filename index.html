Case study:

Supponiamo di sviluppare un software che organizza concerti

la prossima serata sarà dedicata ad un gruppo Rock



Come fare?

DepositoGiordani


public class DepositoGiordani {

 private Band band;

 public DepositoGiordani(){
  this.band = new Band();
 }

 public void claimRockBand(){
  band.playSomething();
 }
}
  • il DepositoGiordani ha bisogno di una Band
  • usa il suo costruttore per istanziarla
